# NVD26
A Python data visualizer built using Python for my Code in Place final project.

Choosing a final project was not an easy decision. After much deliberation, I settled on building a Python data visualizer, not because it was the simplest option, but because it was the most personally meaningful one.

For the past six months, I have been working on a morphometric research paper, spending the most recent months analyzing data in SPSS and writing the manuscript in Microsoft Word. That process forced me to think carefully about which measurements mattered most and how best to communicate them. When it came time to choose a final project, it felt natural to bring that same data into Python and visualize it in a way I had never done before.

The visualizer supports bar charts, pie charts, histograms, scatter plots, line plots, violin plots, box plots, heatmap correlations, and pair plots. Together, these gave me a richer appreciation of the morphometric data I collected, and in several cases, the visualizations helped me refine how I presented findings in my results and discussion sections.

Building this project also meant learning features I had not encountered yet. I used Claude AI for idea generation and implementation guidance, and Microsoft Copilot within VS Code for debugging and code refinement. Both tools helped me move faster and build with more confidence.

This project sits at the intersection of two things I care about: learning to code and doing rigorous research. That is what made it worth building.
